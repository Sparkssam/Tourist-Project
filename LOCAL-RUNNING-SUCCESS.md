## 🐛 If Something Goes Wrong

### Server Crashed?
```bash
# Restart it
npm run dev
```

### Port 3000 Busy?
```bash
# Use different port
npm run dev -- -p 3001
# Then visit: http://localhost:3001
```

### Changes Not Showing?
1. Hard refresh browser: `Ctrl + Shift + R`
2. Clear browser cache
3. Restart dev server: `Ctrl + C`, then `npm run dev`

### Module Errors?
```bash
# Reinstall dependencies
npm install
```
